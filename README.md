# Smartovate-Project

## Description

Application Web Scalable avec CI/CD Azure.

## Technologies

- Frontend : React.js
- Backend : Node.js (Express)
- Base de données : Azure SQL Database
- Cloud : Microsoft Azure
- CI/CD : GitHub Actions
- Gestion de projet : Jira

## Structure

```
frontend/
backend/
```

## Installation

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm start
```
